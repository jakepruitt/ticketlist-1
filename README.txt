README

WINDOWS INSTRUCTIONS
1. Download Rails 4 from http://rubyinstaller.org/downloads/
2. Complete gem install rails
3. Download imagemagick from http://www.imagemagick.org/script/binary-releases.php#windows
4. Put imagemagick in C:/Program Files
3. Open a terminal
4. Complete a bundle install by typing command "bundle install" 
5. Type command "rake db:drop"
6. Type command "rake db:migrate"
7. Run the rails server by typing command "rails server"
8. Run web application on address localhost:3000
